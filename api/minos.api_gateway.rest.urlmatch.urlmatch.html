minos.api_gateway.rest.urlmatch.urlmatch module¶

urlmatch lets you easily check whether urls are on a domain or subdomain

match_pattern = ‘http://.example.com/’

urlmatch(match_pattern, ‘http://subdomain.example.com/’) # True urlmatch(match_pattern, ‘http://sub.subdomain.example.com/’) # True

urlmatch(match_pattern, ‘https://example.com/’) # False urlmatch(match_pattern, ‘http://bad.com/’) # False

<url-pattern> := <scheme>://<host><path> <scheme> := ‘*’ | ‘http’ | ‘https’ <host> := ‘*’ | ‘.’ <any char except ‘/’ and ‘’>+ <path> := ‘/’ <any chars>
